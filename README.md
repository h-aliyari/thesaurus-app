Thesaurus-app

A multi‑platform Flutter application for exploring, searching, and analyzing Islamic sciences terminology.  
This project provides a domain‑based thesaurus with advanced search, semantic relations, and multilingual UI support.


**Features**

- Advanced Search
  - Search across all domains or within a specific domain
  - Domain‑based filtering with instant UI updates
  - “Star search” for viewing all terms inside a domain

- Rich Term Details
  - Lexicon descriptions
  - Related terms
  - Index references
  - Document connections

- Beautiful UI
  - Dynamic gradients for domains
  - Light/Dark theme support
  - Responsive layout for mobile & desktop

- Multilingual Support
  - Persian (FA)
  - English (EN)
  - Arabic (AR)

- Clean Architecture
  - Provider state management
  - Modular API layer
  - Reusable widgets and components



**Project Structure**

lib/
 ├── provider/            # State management (Provider)
 ├── screen/              # Screens & UI pages
 ├── widget/              # Reusable widgets
 ├── service/             # API services & helpers
 ├── model/               # Data models
 └── main.dart            # App entry point



**Getting Started**

1. Clone the repository

git clone https://github.com/USERNAME/REPO-NAME.git
cd REPO-NAME

2. Install dependencies

flutter pub get

3. Run the app

flutter run



**API Configuration**

All API endpoints are defined in:
lib/service/apis.dart

You can update the base URL or endpoints as needed.
